Testing Clean URLs with Live Server

This page demonstrates how the Live Server configuration supports clean URLs without the .html extension.

Note: The links in the navigation bar above use clean URLs (without .html extensions). Click on them to test if the Live Server configuration is working correctly.

How it works:

The .vscode/live-server.json configuration file includes middleware that automatically adds the .html extension to URLs that don't have a file extension. This allows you to use clean URLs in your links while still serving the correct HTML files.

Benefits:

  • Cleaner, more professional URLs
  • Consistent with your production environment
  • No need to modify links when deploying
